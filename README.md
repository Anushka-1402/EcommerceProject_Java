# E-commerce project (Java)

<h3>Technology Stack</h3>
<h5>Backend</h5>
- Advanced Java<br>
- Servlet<br>
- MySQL<br>
- Hibernate

<h5>Frontend</h5>
- JSP<br>
- HTML<br>
- CSS<br>
- Javascript

<h3>Functionalities</h3>

<h5>Admin</h5>
- can view total number of users registered for the website, total number of categories, and the number of products.<br>
- can add new category and new product to the database.
<h5>User</h5>
- can access category wise products.<br>
- maintain a cart.<br>
- needs to have an account for ordering product.
